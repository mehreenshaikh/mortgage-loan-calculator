# mortgage-loan-calculator
This MATLAB program takes the initial principal (term amount) to be repaid, annual interest rate expressed as a decimal, the number of periodic payments per year, and the total number of payment periods (years) during the life of the loan and outputs an amortization table including the payment period, payment, interest paid, principal paid, and outstanding principal. 

# Getting Started
1. Download amortization.m
2. Run amortization.m
3. It will ask you to input the initial principal, annual interest rate, amount of annual periodic payments, and total number of payment periods
